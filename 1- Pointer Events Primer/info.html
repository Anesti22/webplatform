Basic Pointer Events

Pointer Events includes a number of basic events similar to mouse events:

Down and Up

Pointer Events includes basic pointer down and pointer up events. pointerdown is triggered when a user clicks a mouse button or touches the screen with a finger or pen. pointerup is triggered when a user releases a mouse button or releases their finger or pen from touching the screen.
